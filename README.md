# hijikataito.github.io
🐱🐱🐱 hijikataito.com 🐱🐱🐱

## 必要なソフト

nodeをインストールしてください  
https://nodejs.org/ja/

## インストール

```
npm install
```

## コマンド

- PHPサーバの開始 👉 `npm run server` または `npm run server:windows`
- Sassの監視 👉 `npm run watch:css`
- Pugの監視 👉 `npm run watch:pug`

`start.bat` をダボゥクリックで👆のコマンド全部実行
